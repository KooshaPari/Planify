# Support

If you have questions or need help with **Planify**, please use the [GitHub Discussions](https://github.com/KooshaPari/Planify/discussions) for community Q&A.

For bug reports and feature requests, open a [GitHub Issue](https://github.com/KooshaPari/Planify/issues/new/choose).

For security vulnerabilities, see [SECURITY.md](SECURITY.md) — do NOT file a public issue.
