# Planify

Open-source project management tool for issues, cycles, and product roadmaps. Fork of Plane.so adapted for Phenotype ecosystem integration.

## Stack
| Layer | Technology |
|-------|------------|
| Backend | Node.js / TypeScript |
| Frontend | TypeScript / React |
| Package Manager | pnpm (workspace) |
| Container | Docker Compose |
| CI | Renovate, Pre-commit |

## Key Commands
```bash
# Install dependencies
pnpm install

# Development
pnpm dev

# Lint
pnpm lint

# Test
pnpm test
```

## Code Quality
- Linting: ESLint + Prettier
- Formatting: Prettier (via oxfmt)
- Type checking: TypeScript strict mode
